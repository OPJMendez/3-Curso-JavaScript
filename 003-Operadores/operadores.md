**Nos permiten comparar valores**

- **Igual que**

```javascript
// ==
let numberOne, numberTwo;
numberOne = 10;

```

- **Igual en valor y en tipo de dato**

```javascript

numberTwo = 10;
```

- **Diferente en valor de dato**

```javascript
// !=

numberOne = 10;
numberTwo = 20;
if (numberOne != numberTwo) {
  console.log('');
}
```
