        ¿Qué son las variables?
Son espacios en memoria donde podemos guardar información.

En JavaScript:

var -> Se utiliza para variables globales.
let -> Tiene un alcance de tipo bloque. 
const -> Constantes.

Reglas:

-> Deben empezar por letras, $ o un guion bajo ( _ ).

Podemos sobreescribir variables sin importar el tipo de dato.
